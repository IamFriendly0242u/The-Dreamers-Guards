# The Dreamers Guards

**The Dreamers Guards** is a high-performance, enterprise-grade security suite designed to preserve gameplay integrity on Fabric servers. Built specifically for Fabric environments running Java 25, this mod provides a robust defense against unauthorized modifications and exploits.

## Key Features

*   **Advanced Client-Side Validation**: Performs real-time, encrypted network authentication between client and server to verify mod integrity.
*   **Proactive Threat Detection**: Automatically detects and blocks blacklisted modifications via a comprehensive signature-based scanner.
*   **Proactive Discord Integration**: Instant, real-time threat reporting via webhooks, featuring dynamic 3D isometric player head rendering for immediate visual identification.
*   **Multi-Phase Punishment System**: A progressive, 4-stage automated enforcement suite that tracks infraction history and applies adaptive restrictions—ranging from temporary isolation to permanent termination.
*   **Reliable Management**: Features automated configuration backup rotation (storing up to 10 historical backups) to ensure maximum reliability and ease of administration.

## Technical Specifications

*   **Platform**: Fabric Loader
*   **Minecraft Version**: ~26.1.2
*   **Java**: 25+
